# 🍕 Restaurant Page

A small single-page restaurant website built using **JavaScript modules and Webpack**.  
The entire page content is dynamically generated through JavaScript rather than static HTML.

The project focuses on **modular architecture, DOM manipulation, and modern frontend tooling**.

## Live Demo

[View on GitHub Pages](https://inaladevi.github.io/restaurant-page/)

## Features

- **Single Page Application structure** using JavaScript modules.
- **Dynamic DOM Rendering** where each page (Home, Menu, Contact) is generated through JavaScript.
- **Tab Navigation** allowing users to switch between sections without page reloads.
- **Modular Architecture** with separate modules for each page.
- **Webpack Bundling** to bundle JavaScript, CSS, and image assets.
- **Image Asset Management** handled through Webpack imports.
- **Responsive Layout** using CSS Grid and Flexbox.

## Built With

- **HTML5**
- **CSS3**
  - CSS Grid
